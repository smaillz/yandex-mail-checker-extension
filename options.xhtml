For your maximum security, this extension doesn't need your password. To make this extension work, you need to...

  1. You log in - You must log in at normal email website. As long as you stay logged in, it will work. If you log out, it will log out too.
  2. Browser accepts cookies - This extension doesn't need login information but needs cookies.
  3. When the timer that is set checks for new email.

This extension is not spyware so it doesn't know what email you have read. And it cannot remember what is the old or the new one due to the technical problem that comes by not using a username and password. It uses an indirect method to detect the new one by increasing unread counter.

Note. Some emails may be going to both the Inbox and a Custom label. This will make the extension count that email twice. This is one of the limitations of the extension. But if you enable the option 'Use secure email feed', it can help with this issue.

Multiple sign-in is a new addition to Google, not this extension. You need to enable multiple sign-in at Google account settings. If you need more information, please see...

https://support.google.com/accounts/answer/1721977

You don't need this extension to use multiple sign-in. This is Google's own feature. You can make multiple sign-ins from any browser that Google supports. After you enable multiple sign-in at Google, you must also enable multiple sign-in mode in this extension. This will checking up to 3 email accounts instead of only 1 account. Multiple sign-in mode is the same as single sign-in. You need to log in by yourself on all of your accounts.
Feed enhance
Use secure email feed to get unread email count and 3 latest snippets


Show snippets on tooltip


Show snippets on Notification

Secure email feed is a group of https:// URLs that can get the email header and summary. It is an unofficial method and does not appear anywhere in Google help.

Advantage

  • Feed unread counter is more accurate. You will not get the problem of double count when your emails enter both into Inbox and a Custom label.
  • Detecting new email is more accurate because my extension can detect and compare email arrival times.
  • You can get 3 latest snippets via the tooltip and notification.

Disadvantages

  • A faster internet connection is needed. Time out errors may occur easily if you have low internet connection speed or if you enable multiple sign-in mode.
  • Email checking time is longer than usual.
  • Due to it being an unofficial method, it may not work properly in the future.
